# Contributing

PRs are welcome! Here's how to get started:

1. Fork the repo
2. Clone your fork
3. Install dependencies: `npm install`
4. Make your changes
5. Build: `npm run build`
6. Test locally: `npm link` then run `whspr`
7. Submit a PR

## Development

```bash
# Run without building
npm run dev

# Build
npm run build
```

## Ideas for Contributions

- Support for other transcription APIs
- Linux audio input support
- Windows support
- Additional output formats
